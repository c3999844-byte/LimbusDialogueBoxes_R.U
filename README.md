# LimbusDialogueBoxes_R.U

Машинный перевод. Если есть ошибки - сообщите, я исправлю.

Переведенны фразы из "LimbusDialogueBoxes_EN"

Оригинальный автор https://github.com/NotherWael/LimbusDialogueBoxes_EN

Установка... Если вы используете русификатор либо Limbus Localization manager (Без вопросов. Впервые выкладываю что то на github.) 
1. Копируйте скаченный файл BattleSpeechBubbleDlg.
2. Вставляйте его в(это способ для русификтаоров) русификатор и если высвечится кнопка заменить файл, заменяйте.

Возможные проблема:
Вмето слов - кубики. Это проблема в шрифте?... Некоторые шрифты не поддерживают кирилицу, по этому вам надо зайти в русификаторе в папку font и удалить шрифты, и вставить ваш любимый либо же встроенный от переводчиков.
